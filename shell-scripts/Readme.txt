Script Details

1. LogAnalyzer_Startup.sh 
   It will start hadoop cluster and mysql
   To run - ./LogAnalyzer_Startup.sh 

2. LogAnalyzer_Stop.sh  
   It will stop hadoop cluster and mysql
   To run - ./LogAnalyzer_Stop.sh 	

3. Grant.sh
   It will grant the preveligies to acess data from mysql.
   To run - ./Grant.sh

4. Revoke.sh
   It will revoke the granted preveligies.
   To run - ./Revoke.sh

5. ChangePassword.sh
   It will chage hduser's password
   To run - ./ChangePassword.sh

6. mysql.sh
   It will invoke mysql command prompt.
   To run - ./mysql.sh

7. ListGrantedUsers.sh
   It will list all granted users who can acess log data.
   To run - ./ListGrantedUsers.sh
